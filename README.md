# 발달장애 학생 맞춤 인지 가드레일 (연구 버전)

학생 한 명의 인지 세계(텍스트 + 그림 키워드)를 코퍼스로 학습하고,
재난 안내문 등 외부 정보가 그 학생이 이해할 수 있는 범위 안인지
도메인 이탈(=너무 어려움/낯섦)로 검증한다.

## 핵심 설계
- 텍스트 / 그림 키워드는 **각각 선택**. 둘 다 없어도 프로필 성립.
- 그림은 직접 임베딩하지 않고 **키워드 텍스트로 흡수**(CPU 경량 유지).
- 판정/가드레일은 자체 엔진(CoreAIv2Engine). 외부 LLM 불필요.
- 학생 1명 = 프로필 1개 = pkl 1개.

## 파일
- `student_profile.py` — 학생 프로필 코퍼스 (텍스트/그림 각각 선택적)
- `core_ai_v2_engine.py`, `neural_markov_engine.py`, `korean_tokenizer.py` — 엔진
- `app.py` — 교사용 Streamlit UI
- `demo.py` — UI 없이 동작 확인용 (※ 데이터는 전부 예시용 가짜)

## 실행
```
pip install -r requirements.txt
streamlit run app.py     # UI
python demo.py           # 콘솔 데모
```

## 연구 시 주의 (실제 학생 데이터)
- 미성년·발달장애 학생의 발화/그림은 매우 민감한 개인정보.
  보호자 동의, 식별정보 분리 보관 필수.
- 클로바 노트 등 외부 전사 사용 시 "음성이 외부 서버로 전송됨"을 동의서에 명시.
- 전사 결과는 교사가 교정 = 반향어 청크 라벨링 단계로 활용.
- PMI 청크 보존은 데이터가 충분히 쌓여야 의미가 생김(소량이면 우연한 쌍만 잡힘).
